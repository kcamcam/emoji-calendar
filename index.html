What is Emoji Calendar

Add a festive touch to your HTML all year round, simply include a script and add an #id to an element. Emoji Calendar will then automatically insert an festive emoji into the corresponding element depending on the dat or the time of year.

Check it out on GitHub (github.com/emoji-calendar)

How to use Emoji Calendar

1. Give an element on your page the id #emoji-calendar
2. Add the script to the footer of your HTML page
   <script src="https://cdn.jsdelivr.net/gh/kcamcam/emoji-calendar/emoji-calendar.js"></script>
3. Thats all

Note:

Corresponding holidays are official Canadian holidays only.
